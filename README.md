# AI Knowledge Assistant

## 🚀 Overview
This project implements a Retrieval Augmented Generation (RAG) pipeline using LangChain and FAISS to enable document-based question answering.

## ✨ Features
- Upload TXT or PDF documents
- Automatic text chunking and embedding
- Vector search using FAISS
- LLM-based response generation (OpenAI)
- Fallback semantic search without API key
- Interactive UI using Streamlit

## 🧠 Tech Stack
- Python
- LangChain
- FAISS (Vector DB)
- OpenAI / HuggingFace Embeddings
- Streamlit
